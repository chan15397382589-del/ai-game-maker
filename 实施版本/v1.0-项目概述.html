AI Game Classroom

小学生与生成式AI设计伙伴在数字游戏共创中的交互模式研究

实施版本 v1.0 | 2026年6月

项目概述

AI Game Classroom 是一个面向小学三年级学生的AI辅助游戏创作平台。学生通过与AI教师"小智老师"对话，设计并创作HTML5网页游戏。

研究目标

探索小学生与生成式AI在游戏创作中的交互模式，分析不同AI脚手架对计算思维发展的影响。

技术栈

Next.js 16 + React 18 + TypeScript 5 + Tailwind CSS 3.3 + Supabase + 小米MIMO v2.5 Pro

部署平台

Vercel（前端+API）+ Supabase（数据库+认证）

访问地址

www.vibeclassroom.asia

数据库结构

表名 用途 关键字段
users 用户信息 id, name, student_id, role, grade, class_num, srl_condition
messages 聊天消息 id, user_id, role, content, session_id, has_code
conversations 对话文档 id, user_id, title, html_code
game_snapshots 游戏版本 id, user_id, conversation_id, html_code
student_tasks 学生任务 id, user_id, task_id, design_image, game_rules
group_messages 小组消息 id, group_id, user_id, content, message_type
shared_items 分享作品 id, user_id, game_title, html_code
